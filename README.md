# Hello World Flutter App

A clean, modern, and interactive "Hello World" application built with Flutter.

## Features

- **Modern UI**: Styled using Material 3 with a beautiful teal-themed gradient background.
- **Interactivity**: An interactive button that increments a tap counter and updates the greeting state dynamically.
- **Responsive Layout**: Adapts gracefully to all screen sizes.
- **Unit Tested**: Includes widget tests to verify UI rendering and tap interactions.

## Getting Started

### Prerequisites

To run this application, make sure the Flutter SDK is installed and added to your environment path.

If you are using this workspace, you can initialize/re-install dependencies by running:
```bash
./scripts/dev-setup.sh
```

### Running the App

To run the application in development mode:
```bash
flutter run
```

### Running Tests

To run the widget tests:
```bash
flutter test
```
